# 게임 규칙

## 🎮 게임 제목

**Catch Zone (캐치 존)**

---

## 📝 게임 설명

화면을 3개 구역(LEFT, CENTER, RIGHT)으로 나누고, 떨어지는 과일을 바구니로 받아 점수를 획득하는 게임입니다.
폭탄을 피하고 과일을 받아 최대한 높은 점수를 달성하세요!

포즈 인식을 통해 바구니의 위치를 제어합니다:
- **왼쪽 포즈** → LEFT 구역으로 이동
- **정면 포즈** → CENTER 구역으로 이동
- **오른쪽 포즈** → RIGHT 구역으로 이동

---

## 🎯 게임 목표

- 떨어지는 과일을 최대한 많이 받아 높은 점수를 획득하세요!
- 폭탄을 받으면 즉시 게임 오버!
- 과일을 2번 놓치면 게임 오버!
- 목표 점수: 1000점 이상

---

## 🕹️ 게임 규칙

### 기본 규칙

1. **시작**: "Start" 버튼을 클릭하여 게임 시작

2. **화면 구성**:
   - 화면은 좌(LEFT), 가운데(CENTER), 우(RIGHT) 총 3개 구역으로 구성
   - 각 구역의 하단에 바구니가 있음
   - 아이템은 화면 상단에서 세 구역 중 하나로 떨어짐

3. **바구니 제어**:
   - **왼쪽 포즈**: 바구니를 LEFT 구역으로 이동
   - **정면 포즈**: 바구니를 CENTER 구역으로 이동
   - **오른쪽 포즈**: 바구니를 RIGHT 구역으로 이동

4. **아이템 획득**:
   - 바구니와 아이템이 같은 구역에서 만날 때만 획득 판정 발생
   - 과일 아이템을 받으면 점수 증가
   - 폭탄을 받으면 즉시 게임 오버

5. **게임 오버 조건**:
   - 폭탄(Bomb)을 받으면 즉시 게임 오버
   - 과일 아이템을 2번 놓치면 게임 오버

### 놓침(미스) 규칙

- 과일 아이템을 놓치면 다음 규칙 적용:
  - **첫 번째 놓침**: "경고" 표시, 게임은 계속 진행
  - **두 번째 놓침**: 게임 오버

## 🎨 필요한 포즈 목록

게임에 필요한 포즈는 3가지입니다. (Teachable Machine에서 학습할 클래스)

| 포즈 이름 | 설명 | 게임 내 동작 | 예시 |
|---------|------|------------|------|
| 왼쪽 | 얼굴을 왼쪽으로 돌린 상태 | 바구니를 LEFT 구역으로 이동 | 👈 |
| 정면 | 정면을 바라본 중립 상태 | 바구니를 CENTER 구역으로 이동 | 😊 |
| 오른쪽 | 얼굴을 오른쪽으로 돌린 상태 | 바구니를 RIGHT 구역으로 이동 | 👉 |

> **⚠️ 중요**: 현재 학습된 모델에는 "위", "아래" 포즈도 있지만, 이 게임에서는 **"왼쪽", "정면", "오른쪽"** 3가지만 사용합니다.

## ⏱️ 시간 설정

게임 시간 관련 설정:

- **단계별 제한 시간**: 20초 (각 단계마다)
- **아이템 낙하 시간** (단계별 변동):
  - 1단계: 2.0초
  - 2단계: 1.8초
  - 3단계: 1.6초
  - 4단계: 1.4초
  - 5단계 이상: 단계마다 0.2초씩 감소 (최소 0.6초)
- **아이템 생성 간격**: 단계별 낙하 시간의 60%~80% 사이 랜덤 값
  - 예: 1단계(낙하 2초) → 생성 간격 1.2초~1.6초 사이

---

## 🎖️ 단계(Level) 시스템

게임은 여러 단계로 구성되며, 각 단계마다 난이도가 증가합니다.

| 단계 | 단계 시간 | 낙하 시간 | 아이템 생성 간격 | 난이도 |
|-----|---------|---------|---------------|-------|
| 1단계 | 20초 | 2.0초 | 1.2초~1.6초 | 쉬움 |
| 2단계 | 20초 | 1.8초 | 1.08초~1.44초 | 쉬움 |
| 3단계 | 20초 | 1.6초 | 0.96초~1.28초 | 보통 |
| 4단계 | 20초 | 1.4초 | 0.84초~1.12초 | 보통 |
| 5단계 | 20초 | 1.2초 | 0.72초~0.96초 | 어려움 |
| 6단계+ | 20초 | 1.0초 이하 | 더욱 빠름 | 매우 어려움 |

### 단계 진행 규칙

- 게임 오버 없이 단계 시간(20초)이 모두 지나면 다음 단계로 자동 진행
- 단계가 오를 때마다 낙하 시간이 0.2초씩 감소 (최소 0.6초)
- 아이템이 더 빠르게 떨어지고 더 자주 생성되어 난이도 증가

## 💡 게임 개발 팁

### 포즈 인식 매핑

metadata.json의 라벨과 게임 내 사용 매핑:

| metadata.json 라벨 | 게임 내 사용 | 바구니 위치 |
|-------------------|------------|-----------|
| "왼쪽" | LEFT | 왼쪽 구역 |
| "정면" | CENTER | 가운데 구역 |
| "오른쪽" | RIGHT | 오른쪽 구역 |
| "위" | 사용 안 함 | - |
| "아래" | 사용 안 함 | - |

### 게임 구현 시 고려사항

1. **아이템 생성 로직**
   - 3개 구역 중 랜덤하게 선택
   - 폭탄 출현 확률 조절 (예: 20%)
   - 과일 종류도 랜덤 선택

2. **충돌 감지**
   - 아이템의 Y 좌표가 바구니 Y 좌표에 도달했을 때
   - 아이템의 구역과 바구니의 구역이 일치하는지 확인

3. **놓침 판정**
   - 과일 아이템이 화면 하단을 벗어났을 때
   - 폭탄은 놓침으로 카운트하지 않음

4. **단계 전환**
   - 단계별 타이머 관리
   - 단계 시작 시 낙하 속도 및 생성 간격 업데이트
